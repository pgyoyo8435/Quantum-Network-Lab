Protocol lab — 01

BB84 quantum key distribution

Generate qubits, send them through a lossy/noisy fiber channel, let Bob measure in random bases, estimate QBER from a public sample, and compute the remaining secret key after privacy amplification.

🔑 What is BB84? — Expand for a first-principles explanation

The core idea

Alice encodes each bit as a photon polarization, randomly choosing between two incompatible bases: rectilinear (Z: ↑/→) or diagonal (X: ↗/↘). Bob also randomly picks a basis to measure in. When they chose the same basis, Bob's result matches Alice's — these bits are kept. When bases differ, results are random and the bit is discarded. This is called sifting.

Key insight: No physical law can copy an unknown quantum state (no-cloning theorem). Eve must guess the basis, disturbing ~25% of kept bits if she intercepts. This raises the Quantum Bit Error Rate (QBER) above ~11%, signalling her presence.

What to try first

1. Run with default settings — note the QBER and final key length.
2. Slowly increase Optical misalignment and watch QBER rise.
3. Set Eve intercept-resend to 50%+ — the key should abort.
4. Run Monte Carlo to see the statistical distribution of QBER across many trials.
5. Increase fiber distance and observe how fewer detected pulses create noisier QBER estimates.

Quick reference — 05

Glossary of quantum networking

Key terms, physical meanings, and the equations that govern them.

QBER — Quantum Bit Error Rate
The fraction of sifted key bits where Bob's measurement disagrees with Alice's intended bit. The BB84 abort threshold is approximately 11%.
QBER = errors / sifted_bits_tested
Sifting
After quantum transmission, Alice and Bob publicly announce which basis (Z or X) each used. Bits where they chose different bases are discarded.
Privacy Amplification
A classical hashing step that compresses the sifted key, removing any information that Eve might have learned.
f_secret ≈ max(0, 1 − 2·h₂(QBER))
Time-bin Entanglement
Entanglement encoded in the arrival time of photons (early vs late time slots). Robust against polarization drift in fiber.
Franson Interferometer
An unbalanced Mach-Zehnder interferometer used to measure time-bin entanglement. The visibility V directly quantifies entanglement quality.
C(φ) = C₀[1 + V·cos(φ)] / 2
CHSH Inequality & Bell Test
A statistical test of quantum non-locality. The CHSH parameter S = 2√2·V. A measured S > 2 proves genuine quantum entanglement.
S = 2√2 · V
Entanglement Swapping
A Bell-state measurement (BSM) at an intermediate node "teleports" entanglement to the outer nodes, extending it without measuring the qubit value.
F_swap ≈ F₁·F₂ + (1−F₁)·(1−F₂)/3
Spontaneous Raman Scattering
When strong classical light travels through silica fiber, it generates photons at shifted wavelengths via inelastic scattering.
N_Raman ∝ P_classical · L_fiber · Δλ_filter

Recommended learning path

Start with BB84 and run the protocol. Then change one parameter at a time. Run Monte Carlo to see statistical variation.

1. BB84 → learn sifting, QBER, privacy amplification 2. Time-bin pairs → visibility, Bell violation 3. Routing → entanglement swapping, quantum memories 4. Fiber coexistence → Raman noise, WDM power
Model limits and reproducibility

This lab is rigorous as an educational simulator: formulas are explicit, assumptions are logged, controls are reproducible.

Not included in this simulator: - Decoy-state analysis for WCP BB84 - Finite-key security confidence intervals - Detector side-channel attacks

The seed control in BB84 makes runs fully reproducible.
